# 🍲 Recipe App (Flutter)

A simple Recipe App built using Flutter that allows users to view and add recipes.  
This project is built as a beginner-friendly practice app to understand Flutter UI, state management, and dynamic list rendering.

---

## 📱 Features

- 🍽️ View list of recipes
- ➕ Add new recipes
- 🧾 Display ingredients in structured layout
- ⚡ Simple and clean UI
- 🔄 Dynamic list updates using state management

---

## 🛠️ Built With

- Flutter
- Dart
