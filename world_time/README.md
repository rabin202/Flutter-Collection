# 🌍 World Time App

A simple Flutter application that displays the current time in different cities around the world using a World Time API.

## Features

* View current time for multiple cities
* Select a city from a list of locations
* Fetch real-time data from an external API
* Dynamic UI updates based on selected location
* Loading screen while fetching data

## Concepts Practiced

* Flutter Navigation & Route Handling
* Stateful Widgets
* `initState()`
* Async & Await
* HTTP Requests
* JSON Data Handling
* Passing Data Between Screens
* Working with Lists and Custom Objects
