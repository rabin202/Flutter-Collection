# Notes App

A simple and modern Notes Application built with Flutter. The app allows users to create, search, and manage notes while storing data locally using Hive.

## Features

* Create Notes
* View Notes
* Delete Notes
* Search Notes by Keywords
* Local Persistence with Hive
* State Management using Provider
* Swipe to Delete with Slidable
* Responsive UI for multiple screen sizes
* Empty State Illustration using SVG

## Tech Stack

* Flutter
* Dart
* Hive
* Provider
* Flutter Slidable
* Flutter SVG


## State Management

This project uses **Provider** for state management.

Provider handles:

* Notes CRUD operations
* Search functionality
* UI updates
* Hive synchronization
