# 🛍️ Flutter Shop App

A simple Flutter shopping application built as part of my Flutter learning journey.

This project focuses on understanding state management with `setState()`, user interactions, responsive layouts, and basic shopping cart functionality.

## ✨ Features

* Browse products
* Select product size before adding to cart
* Add products to cart
* Remove products from cart
* User feedback using SnackBars
* Size selection using Chips
* Basic responsive UI using LayoutBuilder
* Clean and simple user interface


## 🛠️ Tech Stack

* Flutter
* Dart
