# 🌦️ Weather App

A Flutter-based Weather Application that fetches real-time weather information and forecasts using the OpenWeather API.

---

## 📱 Features
- Displays current temperature
- Shows current weather conditions (Sunny, Cloudy, Rainy, etc.)
- Displays forecast data for upcoming hours

### 📊 Additional Weather Information
- 💧 Humidity
- 🌬️ Wind Speed
- 📌 Atmospheric Pressure

### 🔄 Refresh Weather Data
- Refresh button to fetch the latest weather information

---

## 🛠️ Technologies Used

- Flutter
- Dart
- OpenWeather API
- HTTP Package
- Flutter Dotenv
